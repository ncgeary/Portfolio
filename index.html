NHL 3-2-1 Point System

Personal Project
Tools: Python, BeautifulSoup , NumPy, & Pandas
Github
Live Demo

TLDR: This project is to take a look to see what the NHL's standings would be if they went to a 3-2-1 point system. Brief overview of what the 3-2-1 point system is: 3 points -> Regulation win 2 points -> Overtime/Shootout win 1 point -> Overtime/Shootout loss. To make this happen I am using Python and the BeautifulSoup library to web scrape the NHL standings on hockey-reference.com along with using Numpy to do the calculations needed to find the new standings.
